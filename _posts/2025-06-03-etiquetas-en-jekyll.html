--- layout: post title: "Cómo crear páginas de etiquetas con jekyll-tagging" date: 2025-06-03 tags: [jekyll, etiquetas, plugins] categories: [Tutorial] ---

El plugin jekyll-tagging permite generar automáticamente una página por cada etiqueta.

  • Cada post puede tener múltiples etiquetas
  • Se genera una URL como /tag/nombre/
  • Puedes personalizar el layout de las etiquetas

Es ideal para mejorar la navegación y el SEO de tu blog.
